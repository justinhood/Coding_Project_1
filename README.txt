Within this directory, we construct a routine for preforming a decriptive attack on the Vigenere cipher.
Within the /SRC/ directory is a makefile that will do all of the work for you for compilation.
This code is written in C++, and has no known bugs at this time.

To compile use the make/gmake command within the src directory to compile each of the 4 executables that correspond to each of the stages of the project. 

Further documentation can be found in the enclosed Doxygen mark up, simply open the index.html file from the /html/ directory to see the full markup.

Included in this directory are the respective plaintexts, as proof of functionality of the code.
